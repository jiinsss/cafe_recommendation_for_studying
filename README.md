## item-based-recommendaton
공부하기 좋은 카페를 추천해주는데 공부환경점수를 이용해 sorting 한 것 중 추천해준다

그 중 사용자가 가봤던 혹은 선호하는 카페와 유사한 카페를 아이템 기반으로 추천해주는데

단순 코사인 유사도 분류만 사용하기보다 감성분석을 사용해 걸러낸 긍정 리뷰들 중 유사한 카페를 추천해 줌으로 모델의 정확도를 높였다
